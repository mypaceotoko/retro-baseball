# RETRO BASEBALL

レトロゲーム風のブラウザ野球ゲームです。
伝説の選手たちが集結し、完全アニメーションで9回制の試合を繰り広げます。

## 特徴
- **レトロなピクセルアート風UI**: スコアボードやドット絵風のCanvas描画
- **完全アニメーション**: 投球、スイング、打球の軌道、守備の動き、ランナーの進塁をすべてアニメーションで表現
- **「間」を意識した演出**: 投球→スイング→打球→結果 という野球のリアリティあるテンポを再現
- **実況テキスト**: プレイ状況に応じた熱い実況テキストを表示
- **スマホ対応**: タップ操作に対応、レスポンシブなレイアウト

## 遊び方
1. `index.html` をブラウザで開きます。
2. タイトル画面で「GAME START」をクリック（またはタップ）します。
3. 画面下部の「PITCH !」ボタン、またはグラウンド（Canvas部分）をタップするとピッチャーが投球します。
4. 自動的にバッターがスイングし、結果がアニメーションと実況で表示されます。
5. 3アウトで攻守交代。9回裏までプレイし、勝敗が決まります。

## ファイル構成
- `index.html`: ゲームのメイン画面、UI、スタイリング（CSS）
- `game.js`: ゲームロジック、Canvasアニメーション、選手データ

## GitHub Pagesでの公開手順

このゲームは静的なHTML/CSS/JSのみで構成されているため、GitHub Pagesを使って無料で簡単に公開できます。

1. **GitHubリポジトリの作成**
   - GitHubにログインし、新しいリポジトリを作成します（例: `retro-baseball`）。
   - リポジトリは Public（公開）に設定してください。

2. **ファイルのアップロード**
   - 作成したリポジトリに、本ディレクトリの `index.html` と `game.js` をアップロード（またはプッシュ）します。

3. **GitHub Pagesの有効化**
   - リポジトリの **Settings**（設定）タブを開きます。
   - 左側のメニューから **Pages** を選択します。
   - "Build and deployment" の "Source" を **Deploy from a branch** にします。
   - "Branch" のプルダウンから `main`（または `master`）ブランチを選択し、フォルダを `/ (root)` にして **Save** をクリックします。

4. **公開の確認**
   - 数分待つと、上部に「Your site is live at `https://[ユーザー名].github.io/retro-baseball/`」というURLが表示されます。
   - そのURLにアクセスすると、ゲームをプレイできます！
